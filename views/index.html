Linux

Linux is a family of open source Unix-like operating systems based on the Linux kernel, an operating system kernel first released on September 17, 1991, by Linus Torvalds. Linux is typically packaged in a Linux distribution.

History

The Unix operating system was conceived and implemented in 1969, at AT&T's Bell Laboratories in the United States by four people. First released in 1971, Unix was written entirely in assembly language, as was common practice at the time.
